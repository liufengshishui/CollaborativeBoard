## 项目描述

此项目为基于网页端的多人协作画图的实现。

使用者通过访问网页进入画图界面，进行多人协作画图。

其中主要功能有：
    
    在线画图，提供白板功能和文本功能
    
    画图多样性选择，自定义颜色和画笔大小
    
    多人协作
    
    图片保存及导入
